# 가운데 글자 가져오기

## 1. 문제 설명
단어 s의 가운데 글자를 반환하는 함수, solution을 완성해주세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

### 매개변수
- String s : 단어

### 반환값
- String : 가운데 글자

### 제약조건
- 1 <= s.length() <= 100

## 2. 코드 구현
```java
    public String solution(String s) {
        return s.length() % 2 == 0 ? s.substring(s.length() / 2 - 1, s.length() / 2 + 1) : s.substring(s.length() / 2, s.length() / 2 + 1);
    }
```
