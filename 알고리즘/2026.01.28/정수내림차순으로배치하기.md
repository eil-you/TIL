# 정수 내림차순으로 배치하기

## 1. 문제 설명
정수를 기준으로 내림차순으로 정렬하는 프로그램을 작성해주세요.

### 매개변수
- long num : 정수

### 반환값
- long : num를 내림차순으로 정렬한 배열

### 제약조건
- 0 <= num <=  8000000000

    public long solution(long num) {

        Arrays.sort(ch);
