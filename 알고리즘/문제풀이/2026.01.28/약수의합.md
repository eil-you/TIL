# 약수의 합

## 1. 문제 설명
정수 n명을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

### 매개변수
- int n : 정수

### 반환값
- int : n의 약수를 모두 더한 값

### 제약조건
- 1 <= n <= 3000

## 2. 코드 구현
```java
    public int solution(int n) {
        int answer = 0;
        for (int i = 1; i <= Math.sqrt(n); i++) {
            if(n == i*i) answer+= i;
            else if (n % i == 0) {
                answer += i;
                answer += n / i;
            }   
        }
        return answer;
    }
```
