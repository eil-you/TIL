# [PCCP 기출문제] 3번 - 지폐 접기

## 1.
민수는 다양한 지폐를 수집하는 취미를 가지고 있습니다. 지폐마다 크기가 달라 지갑에 넣으려면 여러 번 접어서 넣어야 합니다. 예를 들어 지갑의 크기가 30 * 15이고 지폐의 크기가 26 * 17이라면 한번 반으로 접어 13 * 17 크기로 만든 뒤 90도 돌려서 지갑에 넣을 수 있습니다. 지폐를 접을 때는 다음과 같은 규칙을 지킵니다.

지폐를 접을 때는 항상 길이가 긴 쪽을 반으로 접습니다.
접기 전 길이가 홀수였다면 접은 후 소수점 이하는 버립니다.
접힌 지폐를 그대로 또는 90도 돌려서 지갑에 넣을 수 있다면 그만 접습니다.
지갑의 가로, 세로 크기를 담은 정수 리스트 wallet과 지폐의 가로, 세로 크기를 담은 정수 리스트 bill가 주어질 때, 지갑에 넣기 위해서 지폐를 최소 몇 번 접어야 하는지 return하도록 solution함수를 완성해 주세요.

지폐를 지갑에 넣기 위해 접어야 하는 최소 횟수를 구하는 과정은 다음과 같습니다.

1. 지갑의 가로, 세로 크기를 담은 정수 리스트 wallet과 지폐의 가로, 세로 크기를 담은 정수 리스트 bill를 순회합니다.
2. 지폐의 가로, 세로 크기를 담은 정수 리스트 bill를 순회합니다.
3. 지폐의 가로, 세로 크기를 담은 정수 리스트 bill를 순회합니다.

### 매개변수
- `wallet`: 지갑의 가로, 세로 크기를 담은 정수 리스트
- `bill`: 지폐의 가로, 세로 크기를 담은 정수 리스트

### 반환값
- 지갑에 넣기 위해서 지폐를 최소 몇 번 접어야 하는지 return

### 제약 조건
- 1 <= wallet.length <= 10
- 1 <= bill.length <= 10
- 1 <= wallet[i] <= 1000
- 1 <= bill[i] <= 1000


## 2. 풀이 전략
1. 지갑과 지폐를 오름차순으로 정렬
2. 반복하며 지폐가 지갑에 들어가는지 확인
3. 안들어가면 큰 값을 절반으로 줄여서 다시 반복문 확인

## 3. 직접 코드 구현(java)
```java
    public int solution(int[] wallet, int[] bill) {
        int answer = 0;
        
        Arrays.sort(wallet);
        
        while(true){
            Arrays.sort(bill);
         
            if(wallet[0] >= bill[0] && wallet[1] >= bill[1]) break;
            
            bill[1] /=2;
            answer++;

        }
        
        return answer;
    }
```
