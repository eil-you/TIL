# 시저 암호

## 1. 문제 설명
어떤 문장의 각 알파벳을 일정한 거리만큼 밀어서 다른 알파벳으로 바꾸는 암호화 방식을 시저 암호라고 합니다. 예를 들어 "AB"는 1만큼 밀면 "BC"가 되고, 3만큼 밀면 "DE"가 됩니다. "z"는 1만큼 밀면 "a"가 됩니다. 문자열 s와 거리 n을 입력받아 s를 n만큼 민 암호문을 만드는 함수, solution을 완성해 보세요.

### 매개변수
- String s : 알파벳으로 이루어진 문자열
- int n : 알파벳을 밀을 거리

### 반환값
- String : s를 n만큼 민 암호문

### 제한 조건
- 공백은 아무리 밀어도 공백입니다.
- s는 알파벳 소문자, 대문자, 공백으로만 이루어져 있습니다.
- s의 길이는 8000 이하입니다.
- n은 1 이상 25 이하인 자연수입니다.


## 2. 문제 풀이
*** String += 은 객체를 계속 생성해야하기 때문에 메모리를 사용하게 된다. 그래서 StringBuilder를 사용***
1. StringBuilder를 사용하여 결과를 저장
2. toCharArray()를 사용하여 문자열을 char 배열로 변환
3. 공백인지 확인
4. 알파벳이 소문자 인지 대문자인지 확인
5. 알파벳이 소문자인 경우 n만큼 밀어서 결과에 추가
6. 알파벳이 대문자인 경우 n만큼 밀어서 결과에 추가
7. 결과를 문자열로 변환하여 반환


## 3. 코드 구현
```java
    public String solution(String s, int n){
        String answer = "";

        // 답변 넣을 SringBuilder-> String은 += 할때 마다 객체 생성필요
        StringBuilder sb = new StringBuilder();


        for(char c : s.toCharArray()){

            if(c == ' '){
                sb.append(' ');
                continue;
            }else{
                char base = Character.isUpperCase(c) ? 'A' : 'a';
                // 핵심 공식 : (현재 문자 -시작문자 + 밀기) % 26 + 시작문자
                char pushed = (char) ((c-base+n)%26+base);
                sb.append(pushed);
            }
        }

        answer = sb.toString();
        return answer;
    }
```
