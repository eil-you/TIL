# [PCCP 기출문제] 12. 추억 점수

## 1. 문제 설명(Map 사용)
사진들을 보며 추억에 젖어 있던 루는 사진별로 추억 점수를 매길려고 합니다. 사진 속에 나오는 인물의 그리움 점수를 모두 합산한 값이 해당 사진의 추억 점수가 됩니다. 예를 들어 사진 속 인물의 이름이 ["may", "kein", "kain"]이고 각 인물의 그리움 점수가 [5점, 10점, 1점]일 때 해당 사진의 추억 점수는 16(5 + 10 + 1)점이 됩니다. 다른 사진 속 인물의 이름이 ["kali", "mari", "don", "tony"]이고 ["kali", "mari", "don"]의 그리움 점수가 각각 [11점, 1점, 55점]]이고, "tony"는 그리움 점수가 없을 때, 이 사진의 추억 점수는 3명의 그리움 점수를 합한 67(11 + 1 + 55)점입니다.

그리워하는 사람의 이름을 담은 문자열 배열 name, 각 사람별 그리움 점수를 담은 정수 배열 yearning, 각 사진에 찍힌 인물의 이름을 담은 이차원 문자열 배열 photo가 매개변수로 주어질 때, 사진들의 추억 점수를 photo에 주어진 순서대로 배열에 담아 return하는 solution 함수를 완성해주세요.


### 매개변수
- name : String[]
- yearning : int[]
- photo : String[][]

### 반환값
- int[]

### 제약 조건
- 1 <= name.length <= 100
- 1 <= yearning.length <= 100
- 1 <= photo.length <= 50
- 1 <= photo[i].length <= 20
- name[i]는 알파벳 소문자로 이루어져 있습니다.
- yearning[i]는 1 이상 100 이하의 정수입니다.
- photo[i][j]는 알파벳 소문자로 이루어져 있습니다.


## 2. 풀이 전략
1. Map<String, Integer>를 사용하여 name과 yearning을 저장합니다.
2. photo를 순회하며 각 사진 포인트 계산
-  map.getOrDefault() 사용하여 포인트 계산
3. photo 포인트를 배열에 담아 return


## 3. 직접 코드 구현 (Java)
```java
import java.util.*;

class Solution {
    public int[] solution(String[] name, int[] yearning, String[][] photo) {
        
        int[] answer = {}

        int n = name.length;
        
        Map<String, Integer> map = new HashMap<>();
        for(int i = 0; i < n; i++){
            map.put(name[i], yearning[i]);
        }

        for(int i = 0; i< photo.length; i++){
            int point = 0;
            for(int j = 0; j < photo[i].length; j++){
                point += map.getOrDefault(photo[i][j], 0);
            }
            answer[i] = point;
        }

        return answer;
    }
}
```
