# 최댓값과 최솟값

## 1. 문제 설명
문자열 s에는 공백으로 구분된 숫자들이 저장되어 있습니다. str에 나타나는 숫자 중 최소값과 최대값을 찾아 이를 "(최소값) (최대값)"형태의 문자열을 반환하는 함수, solution을 완성하세요.
예를들어 s가 "1 2 3 4"라면 "1 4"를 리턴하고, "-1 -2 -3 -4"라면 "-4 -1"을 리턴하면 됩니다.

### 매개변수
- String s : 공백으로 구분된 숫자 문자열

### 반환값
- String : "(최소값) (최대값)" 형태의 문자열

### 제한 사항
- s에는 -1,000,000 이상 1,000,000 이하의 정수가 공백으로 구분되어 저장되어 있습니다.
- s에는 적어도 한 개 이상의 정수가 존재합니다.
- s에는 공백으로 구분된 숫자들이 저장되어 있습니다.

### 입출력 예
| s | result |
| --- | --- |
| "1 2 3 4" | "1 4" |
| "-1 -2 -3 -4" | "-4 -1" |
| "1 2 -3 4 5" | "-3 5" |

## 2. 문제 풀이
1. String.split()을 사용하여 공백을 기준으로 문자열을 분리합니다.
2. Integer.parseInt()로 string을 숫자로 변환
3. min = Integer.MAX_VALUE, max = Integer.MIN_VALUE로 초기화
4. for문을 돌면서 min과 max를 갱신
5. "min + " " + max"를 반환

## 3. 코드 구현
```java
 class solution{
    public String solution(String[] s){
        String[] parts = s.split(" ");
        int[] numbers = new int[parts.length];

        for(int i = 0; i<parts.length ; i++){
            numbers[i] = Integer.parseInt(parts[i]);
        }

        int min = Integer.MAX_VALUE;
        int max = Integer.MIN_VALUE;

        for(int number : numbers){
            min = Math.min(min, number);
            max = Math.max(max, number);
        }

        return min+" "+max;
    }
 }
```
