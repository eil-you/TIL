# 자연수 뒤집어 배열로 만들기

## 1. 문제 설명
자연수 n을 뒤집어 각 자리 숫자를 원소로 가지는 배열 형태로 리턴해주세요. 예를 들어 n이 12345이면 [5,4,3,2,1]을 리턴합니다.

### 매개변수 
- long n : 자연수 

### 반환값 
- int[] : n을 뒤집어 각 자리 숫자를 원소로 가지는 배열 

### 제약조건 
- 0 <= n <= 10,000,000,000

## 2. 코드 구현
```java
    public int[] solution(long n) {
        int[] answer = new int[String.valueOf(n).length()];
        int index = 0;
        while (n > 0) {
            answer[index++] = (int) (n % 10);
            n /= 10;
        }
        return answer;
    }
``` 



## 3. 다른 풀이(StringBuilder 사용)
```java
    public int[] solution(long n) {
        // 1. 숫자를 문자열로 변환 후 뒤집기
        String s = new StringBuilder(String.valueOf(n)).reverse().toString();

        // 2. 문자열을 배열로 변환
        int[] answer = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            answer[i] = s.charAt(i) - '0';
        }
        return answer;
    }
```
