# 다트게임

## 1. 문제 설명
카카오톡 게임별의 하반기 신규 서비스로 다트 게임을 출시하기로 했다. 다트 게임은 다트판에 다트를 세 차례 던져 그 점수의 합계로 실력을 겨루는 게임으로, 모두가 간단히 즐길 수 있다.
갓 입사한 무지는 코딩 실력을 인정받아 게임의 핵심 부분인 점수 계산 로직을 맡게 되었다. 다트 게임의 점수 계산 로직은 아래와 같다.

다트 게임은 총 3번의 기회로 구성된다.
각 기회마다 얻을 수 있는 점수는 0점에서 10점까지이다.
점수와 함께 Single(S), Double(D), Triple(T) 영역이 존재하고 각 영역 당첨 시 점수에서 1제곱, 2제곱, 3제곱 (점수1 , 점수2 , 점수3 )으로 계산된다.
옵션으로 스타상(*) , 아차상(#)이 존재하며 스타상(*) 당첨 시 해당 점수와 바로 전에 얻은 점수를 각 2배로 만든다. 아차상(#) 당첨 시 해당 점수는 마이너스된다.
스타상(*)은 첫 번째 기회에서도 나올 수 있다. 이 경우 첫 번째 스타상(*)의 점수만 2배가 된다. (예제 4번 참고)
스타상(*)의 효과는 다른 스타상(*)의 효과와 중첩될 수 있다. 이 경우 중첩된 스타상(*) 점수는 4배가 된다. (예제 4번 참고)
스타상(*)의 효과는 아차상(#)의 효과와 중첩될 수 있다. 이 경우 중첩된 아차상(#)의 점수는 -2배가 된다. (예제 5번 참고)
Single(S), Double(D), Triple(T)은 점수마다 하나씩 존재한다.
스타상(*), 아차상(#)은 점수마다 둘 중 하나만 존재할 수 있으며, 존재하지 않을 수도 있다.
0~10의 정수와 문자 S, D, T, *, #로 구성된 문자열이 입력될 시 총점수를 반환하는 함수를 작성하라.


### 매개변수
- String dartResult : dartResult

### 반환값
- int : dartResult 3번의 기회에서 얻은 점수 합계

### 제약 조건
- dartResult의 길이는 1 이상 100 이하입니다.
- dartResult는 0~10의 정수와 문자 S, D, T, *, #로 구성된 문자열입니다.


## 2. 문제 풀이
1. isDigit()를 사용하여 dartResult[i]가 숫자인지 확인
2. dartResult[i]가 숫자라면 dartResult[i]를 score에 저장
    - dartResult[i]가 10이면 score를 10으로 저장
3. dartResult[i]가 S라면 score를 score에 저장
4. dartResult[i]가 D라면 score를 score*2에 저장
5. dartResult[i]가 T라면 score를 score*3에 저장
6. dartResult[i]가 *라면 score를 score*2에 저장
7. dartResult[i]가 #라면 score를 score*-1에 저장

## 3. 코드 구현
```java
    class Solution {
        public int solution(String dartResult) {
            int answer = 0;

            int[] score = new int[3];

            int index = -1;

            for(int i = 0; i<dartResult.length(); i++){
                if(Character.isDigit(dartResult.charAt(i))){
                    index++;
                    if(dartResult.charAt(i) == '1' && dartResult.charAt(i+1) == '0'){
                        score[index] = 10;
                        i++;
                    }else{
                        score[index] = dartResult.charAt(i) - '0';
                    }
                }else if(dartResult.charAt(i) == 'S'){
                    score[index] = score[index];
                }else if(dartResult.charAt(i) == 'D'){
                    score[index] = score[index]*score[index]*score[index];
                }else if(dartResult.charAt(i) == 'T'){
                    score[index] = score[index]*score[index]*score[index]*score[index];
                }else if(dartResult.charAt(i) == '*'){
                    score[index] = score[index]*2;
                    if(index > 0){
                        score[index-1] = score[index-1]*2;
                    }
                }else if(dartResult.charAt(i) == '#'){
                    score[index] = score[index]*-1;
                }
                answer += score[index];
            }
            
            return answer;
        }
    }
```

## 4. 다른 풀이(정규식 패턴)
```java
    class Solution {
        public int solution(String dartResult) {
            int answer = 0;
            Matcher matcher = Pattern.compile("([0-9]+)([SDT])([*#]?)").matcher(dartResult);
            int idx = 0;

            while(matcher.find()){
                // 1. 점수 계산
                int num = Integer.parseInt(matcher.group(1));
                String bonus = matcher.group(2);
                String option = matcher.group(3);

                // 2. 보너스 적용
                if(option.equals("D")) s = (int) Math.pow(num, 2);
                else if(option.equals("T")) s = (int) Math.pow(num, 3);
 
                score[idx] = s;

                // 3. 옵션적용
                if(option.equals("*")){
                    score[idx] *= 2;
                    if(idx >0) score[idx-1] *= 2;
                }else if(option.equals("#")){
                    score[idx] *= -1;
                }
                idx++;
            }
            

            return score[0] + score[1] + score[2];
        }
    }
```
