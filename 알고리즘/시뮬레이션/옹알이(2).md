# [PCCP 기출문제] 31. 옹알이 (2)

## 1. 문제 설명
머쓱이는 태어난 지 11개월 된 조카를 돌보고 있습니다. 조카는 아직 "aya", "ye", "woo", "ma" 네 가지 발음과 네 가지 발음을 조합해서 만들 수 있는 발음밖에 하지 못하고 연속해서 같은 발음을 하는 것을 어려워합니다. 문자열 배열 babbling이 매개변수로 주어질 때, 머쓱이의 조카가 발음할 수 있는 단어의 개수를 return하도록 solution 함수를 완성해주세요.

### 매개변수 
- babbling : String[]

### 반환값
- int

### 제한 조건
- 1 <= babbling.length <= 100
- 1 <= babbling[i].length <= 30
- babbling[i]는 알파벳 소문자로 이루어진 문자열입니다.

## 2. 문제 풀이
1. 중복된 발음은 제외
- contains()를 사용해서 제거
2. replace()를 사용해서 발음을 제거
3. 남아있는 발음이 없으면 count++

## 3. 코드 구현
```java
    public int solution(String[] babbling) {
        int answer = 0;

        for(String b : babbling){

            if(b.contains("ayaaya") || b.contains("yeye") || b.contains("woowoo") || b.contains("mama")){
                continue;
            }

            b = b.replace("aya", "");
            b = b.replace("ye", "");
            b = b.replace("woo", "");
            b = b.replace("ma", "");

            if(b.length() == 0){
                answer++;
            }

        }
        return answer;
    }
```
