# 3진법뒤집기

## 1. 문제 설명
자연수 n이 매개변수로 주어집니다. n을 3진법 상에서 앞뒤로 뒤집은 후, 이를 다시 10진법으로 표현한 수를 return 하도록 solution 함수를 완성해주세요

### 매개변수
- n : int

### 반환값
- int

### 제약 조건
- n은 1이상 100,000이하의 자연수입니다.

## 2. 문제 풀이

## 3. 코드 구현(java)
```java
    public int solution(int n) {
        int answer = 0;
       
        String ternary = "";

        while (n > 0) {
            ternary = (n % 3) + ternary;
            n /= 3;
        }

        answer = Integer.parseInt(ternary, 3);
        return answer;
    }
```
