## 3. 직접 코드 구현 (Java)
```java
    public long solution(int[] a, int[] b) {
        long answer = 0;
        for (int i = 0; i < a.length; i++) {
            answer += a[i] * b[i];
        }
        return answer;
    }
```

## 4. 다른 코드
```java
    public long solution(int[] a, int[] b) {
        return IntStream.range(0, a.length).map(i -> a[i] * b[i]).sum();
    }
```
