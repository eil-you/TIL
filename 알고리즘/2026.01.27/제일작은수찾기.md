# 제일 작은 수 찾기

## 1. 문제 설명

이 문제에는 표준 입력으로 정수 배열 numbers가 주어집니다.
numbers에서 최소값과 최대값을 찾아서 차이를 구하는 함수, solution을 완성해주세요.

### 매개변수
- int[] arr : 정수 배열

### 반환값
- int[] : numbers의 최솟값

### 제약조건
- 1 <= arr.length

## 2. 코드 구현
```java
    public int[] solution(int[] arr) {
       
        // 1. 빈배열이거나 요소가 하나면 -1 리턴
        if(arr.length <=1) return new int[]{-1};
        
        // 2. 최솟값 찾기
        int min = arr.Stream().min().getAsInt();

        // 3. 최솟값만 제외하고 다시 배열로 만들기
        return Arrays.stream(arr).filter(x -> x != min).toArray();
    }
```
