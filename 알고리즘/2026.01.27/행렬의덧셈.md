# 행렬의 덧셈

## 1. 문제 설명

이 문제에는 표준 입력으로 두 개의 N×M 행렬이 주어집니다.
각 행렬의 같은 위치에 있는 숫자들을 더한 결과를 가진 N×M 행렬을 리턴하는 함수, solution을 완성해주세요.

### 매개변수
- int[][] arr1 : N×M 행렬
- int[][] arr2 : N×M 행렬

### 반환값
- int[][] : 두 행렬의 합

### 제약조건
- arr1, arr2 행과 열 < 500

## 2. 코드 구현
```java
    public int[][] solution(int[][] arr1, int[][] arr2) {
        int[][] answer = new int[arr1.length][arr1[0].length];
        for (int i = 0; i < arr1.length; i++) {
            for (int j = 0; j < arr1[i].length; j++) {
                answer[i][j] = arr1[i][j] + arr2[i][j];
            }
        }
        return answer;
    }

```
