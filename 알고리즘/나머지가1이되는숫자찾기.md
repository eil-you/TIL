# [PCCP 기출문제] 19. 나머지 1

## 1. 문제 설명
자연수 n이 매개변수로 주어집니다. n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 return 하도록 solution 함수를 완성해주세요. 답이 항상 존재함은 증명될 수 있습니다.

### 매개변수
- n : int

### 반환값
- int

### 제약조건
- 1 <= n <= 1000000

## 2. 문제 풀이


## 3. 직접 코드 구현 (Java)
```java
    public int solution(int n) {
        for (int i = 2; i < n; i++) {
            if (n % i == 1) {   
                return i;
            }
        }
        return -1;
    }
```
