# [PCCP 기출문제] 10. 데이트 분석

## 1. 문제 설명(Map 사용)
AI 엔지니어인 현식이는 데이터를 분석하는 작업을 진행하고 있습니다. 데이터는 ["코드 번호(code)", "제조일(date)", "최대 수량(maximum)", "현재 수량(remain)"]으로 구성되어 있으며 현식이는 이 데이터들 중 조건을 만족하는 데이터만 뽑아서 정렬하려 합니다.

예를 들어 다음과 같이 데이터가 주어진다면

data = [[1, 20300104, 100, 80], [2, 20300804, 847, 37], [3, 20300401, 10, 8]]
이 데이터는 다음 표처럼 나타낼 수 있습니다.

code	date	maximum	remain
1	20300104	100	80
2	20300804	847	37
3	20300401	10	8
주어진 데이터 중 "제조일이 20300501 이전인 물건들을 현재 수량이 적은 순서"로 정렬해야 한다면 조건에 맞게 가공된 데이터는 다음과 같습니다.

data = [[3,20300401,10,8],[1,20300104,100,80]]
정렬한 데이터들이 담긴 이차원 정수 리스트 data와 어떤 정보를 기준으로 데이터를 뽑아낼지를 의미하는 문자열 ext, 뽑아낼 정보의 기준값을 나타내는 정수 val_ext, 정보를 정렬할 기준이 되는 문자열 sort_by가 주어집니다.

data에서 ext 값이 val_ext보다 작은 데이터만 뽑은 후, sort_by에 해당하는 값을 기준으로 오름차순으로 정렬하여 return 하도록 solution 함수를 완성해 주세요. 단, 조건을 만족하는 데이터는 항상 한 개 이상 존재합니다.

### 매개변수
- `data`: ["코드 번호(code)", "제조일(date)", "최대 수량(maximum)", "현재 수량(remain)"]으로 구성된 2차원 정수 배열
- `ext`: 정보를 추출할 기준이 되는 문자열 ("code", "date", "maximum", "remain" 중 하나)
- `val_ext`: 정보를 추출할 기준값 (해당 컬럼의 값이 `val_ext`보다 작은 데이터만 추출)
- `sort_by`: 정보를 정렬할 기준이 되는 문자열 ("code", "date", "maximum", "remain" 중 하나)

### 반환값
- 조건에 맞는 데이터를 `sort_by` 기준으로 오름차순 정렬한 2차원 정수 배열

### 제약 조건
- 1 <= `data`의 길이 <= 500
- 1,000,000 <= `date` <= 2,000,000
- 1 <= `code`, `maximum`, `remain` <= 100,000
- 모든 데이터의 `code` 값은 서로 다릅니다.
- `ext`와 `sort_by`는 항상 등재된 네 가지 문자열 중 하나입니다.

## 2. 풀이 전략
1. **컬럼 매핑**: 문자열로 주어지는 컬럼명(`ext`, `sort_by`)을 배열 인덱스(0~3)로 변환합니다.
2. **필터링**: `data`를 순회하며 `ext` 컬럼값이 `val_ext`보다 작은 것만 리스트에 담습니다.
3. **정렬**: 리스트를 `sort_by` 컬럼 기준으로 오름차순 정렬합니다.
4. **변환**: 리스트를 다시 2차원 배열로 변환하여 반환합니다.

## 3. 직접 코드 구현 (Java)
```java
import java.util.*;

class Solution {
    public int[][] solution(int[][] data, String ext, int val_ext, String sort_by) {
        // 1. 컬럼명에 따른 인덱스 매핑
        Map<String, Integer> colMap = new HashMap<>();
        colMap.put("code", 0);
        colMap.put("date", 1);
        colMap.put("maximum", 2);
        colMap.put("remain", 3);
        
        int extIdx = colMap.get(ext);
        int sortIdx = colMap.get(sort_by);
        
        // 2. 필터링 및 리스트 생성
        List<int[]> filteredData = new ArrayList<>();
        for (int[] d : data) {
            if (d[extIdx] < val_ext) {
                filteredData.add(d);
            }
        }
        
        // 3. 정렬 (sort_by 기준)
        Collections.sort(filteredData, (a, b) -> a[sortIdx] - b[sortIdx]);
        
        // 4. 리스트를 2차원 배열로 변환
        int[][] answer = new int[filteredData.size()][4];
        for (int i = 0; i < filteredData.size(); i++) {
            answer[i] = filteredData.get(i);
        }
        
        return answer;
    }
}   
```
