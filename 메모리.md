# 자바의 메모리
- 자바 실행시 메모리 변화를 알고 객체지향언어의 특징들과의 관계도 알아보자.


## java 프로그램 실행 
JDK > JRM > JVM
- JDK : JAVA DEVELOPMENT KIT(자바 개발 도구)
- JRM : JAVA RUNTIME ENVIROMENT(자바 실행 환경)
- JVM : JAVA VIRTUAL MACHINE(자바 가상 기계)
